# Contributing

Grazie per il tuo interesse a contribuire a ristorantemoka!

Linee guida:
- Forka il repository.
- Crea un branch per la tua modifica, es.: `git checkout -b feature/nome-della-feature`.
- Scrivi codice chiaro e documenta le tue modifiche.
- Aggiungi o aggiorna i test quando applicabile.
- Apri una Pull Request descrivendo cosa cambi e perché; collega l'issue correlata se presente.
- Mantieni il codice conforme allo stile del progetto e assicurati che la CI passi.

Grazie per il contributo!
